# vue-ssr-starter-kit

> A Vue.js project with vue 2.0, vuex and Server Side Rendering. No Vue Routing

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```
